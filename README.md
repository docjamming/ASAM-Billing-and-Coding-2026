# ASAM-Billing-and-Coding-2026
Central Repository for ASAM 2026 Billing and Coding Workshop

Created by a cohort of clinical professional from Cooper University and RWJ Barnabas to serve as a helpful guide for addiction medicine providers to learn tools to convert clinical effort to clincial reimbursements with major payors 

Please ensure that you confirm any clincial approaches with local billing and coding experts prior to implementing them in your own practice location
